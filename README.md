# SpotifySample

### Spotify APIの認証情報の取得
このアプリを動かすには、SpotifyのDeveloper Dashboardでアプリを登録し、認証キーを取得する必要があるります。

- Spotify for Developers にアクセスしてログイン
- 「Create app」から新しいアプリを作成（App nameなどは適当でOK）
- 利用するAPI/SDKを指定するチェックボックスは「Web API」を選択
- 「Redirect URIs」の項目がありますが、この値は適当
- アプリのダッシュボード設定（Settings）から、Client ID と Client Secret を使用する
